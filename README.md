16、运行python文件(eg:module1.py)的三种方法  
*作为程序运行：$ python module1.py  
*作为模块运行：1. $ python 2. >>> import module1  
*作为脚本运行：1. 在python文件首行增加代码：#!/usr/bin/python 2. 给文件赋予执行权限：chmod +x module1.py 3.执行脚本文件./module.py
